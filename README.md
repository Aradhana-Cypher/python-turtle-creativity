# 🐢 Python Turtle Creative Projects

A collection of creative and visual programs built using Python's `turtle` module and `random` library — exploring art, math, and animation through code.

---

## 🗂️ Projects Included

| File | Description |
|------|-------------|
| `Calender_Clock.py` | Displays a real-time clock and calendar using turtle graphics |
| `flower_pattern_design.py` | Generates beautiful geometric flower designs using loops and angles |
| `Fractal_tree.py` | Draws a recursive fractal tree using recursion concepts |
| `Galaxy_effect.py` | Simulates a galaxy effect using turtle + random library |
| `heart_shape_art.py` | Draws a heart shape using mathematical curves |

---

## 🛠️ Requirements
Python 3.x, turtle (built-in), random (built-in)

## ▶️ How to Run

python filename.py

## 💡 About
These projects were built to explore creative coding — combining mathematics, recursion, and randomness to produce visual art through programming
